# Amazon-Sales-Excel-Dashboard
Excel dashboard analyzing Amazon sales performance, regional trends, delivery metrics, and key business KPIs.

## Project Overview

This project analyzes Amazon sales data using Microsoft Excel. The dashboard provides insights into sales performance, regional trends, product categories, delivery efficiency, and key business KPIs.

## Tools Used

- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- KPI Cards
- Data Cleaning
- Conditional Formatting

## Key Metrics

- Total Revenue
- Effective Sales
- Order Status Analysis
- Delivery Performance
- Regional Sales Distribution
- Product Category Performance

## Dashboard Features

- Interactive Filters
- KPI Tracking
- Regional Performance Analysis
- Product Analysis
- Delivery Time Analysis
- Sales Trend Visualization
